Enable type editing on a selected schema

Submit a changed type with current text input value
Enter
Cancel editing

Changeable types are: record, list, tuple, string, number, boolean, null, union, any,

Ref type, type in the defined top level model name.

Value type, type in any json value (e.g. "a", 1, false, null), array and object json is not allowed (for now).
