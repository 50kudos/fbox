Enter
submit a changed key with current text input value

Change type

shift + Enter
Enable type editing on a selected schema
Enter
Submit a changed type with current text input value
Escape
Cancel editing
Note: Changeable types are: record, list, tuple, string, number, boolean, null, union, any.

For Ref type, type in the defined top level model name.

For Value type, type in any json value (e.g. "a", 1, false, null), array and object json is not allowed (for now).
